App for easy management game servers


Supported features:
- Servers monitoring
- Player Details
- Banning, kicking and muting players
- Server map change
- RCON with autocomplete and shortcuts
- Detailed server info
- Used resources by server
- View and change server cvars
- OGP, CW-SERV.RU, CSHost.com.ua and Myarena.ru (Console, stop/start server, extra data)
- Version check
- Widgets
